# 图片压缩工具

一个现代化的在线图片压缩工具，采用苹果风格设计，提供高效、易用的图片压缩服务。

## 功能特性

### 核心功能
- **多格式支持**: 支持 PNG、JPG、WebP 等主流图片格式
- **批量处理**: 可同时上传多张图片进行批量压缩
- **实时预览**: 压缩前后图片对比预览，直观查看压缩效果
- **智能压缩**: 基于浏览器端压缩，保护用户隐私
- **一键下载**: 支持单张或批量下载压缩后的图片

### 压缩设置
- **质量调节**: 0.1-1.0 可调节压缩质量
- **尺寸限制**: 可设置最大宽度和高度
- **性能优化**: 支持 Web Worker 后台处理
- **实时反馈**: 显示压缩进度和状态

### 用户体验
- **拖拽上传**: 支持拖拽文件到页面进行上传
- **响应式设计**: 完美适配桌面端和移动端
- **苹果风格**: 采用苹果设计语言，界面简洁美观
- **实时统计**: 显示压缩率、节省空间等统计信息

## 技术栈

### 前端技术
- **React 18**: 现代化的前端框架
- **Vite**: 快速的构建工具
- **Tailwind CSS**: 实用优先的CSS框架
- **Lucide React**: 精美的图标库

### 核心库
- **browser-image-compression**: 浏览器端图片压缩库
- **Web Workers**: 后台处理，避免界面卡顿

### 开发工具
- **ESLint**: 代码质量检查
- **PostCSS**: CSS后处理器
- **Autoprefixer**: CSS前缀自动添加

## 使用说明

### 基本操作
1. **上传图片**: 点击上传区域或拖拽图片文件到页面
2. **调整设置**: 根据需要调整压缩质量和尺寸限制
3. **开始压缩**: 点击"压缩"按钮或"压缩全部"按钮
4. **预览效果**: 查看压缩前后的对比效果
5. **下载文件**: 点击下载按钮保存压缩后的图片

### 压缩设置说明
- **压缩质量**: 数值越高，图片质量越好，文件越大
- **最大宽度/高度**: 限制图片的最大尺寸，超出部分会被缩放
- **Web Worker**: 启用后台处理，避免界面卡顿

### 支持的文件格式
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

### 文件大小限制
- 单个文件最大: 50MB
- 同时处理文件数: 最多10张

## 常见问题

### Q: 为什么压缩后的图片质量下降了？
A: 压缩会损失一定的图片质量，这是为了减小文件大小。您可以通过调整压缩质量设置来平衡文件大小和图片质量。

### Q: 支持哪些图片格式？
A: 目前支持 JPEG、PNG 和 WebP 格式。如果上传其他格式的文件，系统会提示错误。

### Q: 压缩后的图片会保存在服务器上吗？
A: 不会。所有图片处理都在您的浏览器中进行，不会上传到服务器，保护您的隐私。

### Q: 可以同时处理多少张图片？
A: 最多可以同时处理10张图片，这是为了避免浏览器性能问题。

### Q: 压缩需要多长时间？
A: 压缩时间取决于图片大小和您的设备性能。通常几秒钟内就能完成。

---

**注意**: 这是一个纯前端应用，所有图片处理都在浏览器端完成，不会上传到服务器，保护用户隐私。
